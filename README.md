# 🚗 Car Price Prediction

A Machine Learning web application that predicts the price of used cars based on various features like company, model, fuel type, year, and kilometers driven.

---

## 📌 Overview

This project uses Machine Learning algorithms to estimate car prices accurately.  
The application is built using Flask and deployed with a simple and interactive user interface.

---

## ✨ Features

- Predict used car prices instantly
- Simple and clean UI
- Machine Learning based prediction
- Data preprocessing and analysis
- Flask web application

---

## 🛠️ Tech Stack

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML/CSS
